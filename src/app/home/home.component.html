A Modern CSS Utility Library

CSSFusion is a versatile and lightweight CSS utility library distributed via npm. It provides a rich set of prebuilt classes and customizable tools to simplify and speed up the styling of your web projects.

Explore the Modules in CSSFusion

CSSFusion provides everything you need to create stylish, modern, and responsive web applications. Below are the main modules available in the project:

Components

Build your UI effortlessly with pre-styled components like accordions, buttons, dropdowns, and more. Simply copy the provided HTML and, for components with functionality like dropdowns, include the accompanying TypeScript files.

Templates

Assemble beautiful templates using CSSFusion components. These templates are pre-designed with SCSS variables, allowing you to customize and apply different themes seamlessly to your application.

Customizer

Customize components like accordions or buttons with a Figma-inspired toolset. Select any component and style it to fit your vision with text formatting, shadow effects, dimensions, and more.
